# `htmlValidTags`

Retorna uma lista de tags HTML válidas que podem ser utilizadas em um documento.

## Assinatura da Função

```typescript
export const htmlValidTags = (): string[];
```

## Parâmetros

Esta função não recebe parâmetros.

## Retorno

- **string[]**:  
  Um array de strings, onde cada string representa uma tag HTML válida.

## Exemplo de Uso

import { htmlValidTags } from 'caminho-do-seu-arquivo-de-utilitarios';

const validTags = htmlValidTags();

## Descrição

A função `htmlValidTags` retorna uma lista de todas as tags HTML que são consideradas válidas em um documento HTML. Essa função é útil quando se deseja validar ou filtrar tags em uma string HTML para evitar o uso de tags não suportadas ou potencialmente perigosas.
