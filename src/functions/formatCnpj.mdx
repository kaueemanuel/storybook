# `formatCnpj`

Formata uma string de CNPJ (Cadastro Nacional da Pessoa Jurídica) no padrão brasileiro.

## Assinatura da Função

```typescript
export const formatCnpj = (str: string): string;
```

## Parâmetros

- **str**: `string`  
  A string que representa o CNPJ que você deseja formatar.

## Retorno

- **string**:  
  O CNPJ formatado no padrão brasileiro.

## Exemplo de Uso

```tsx
import { formatCnpj } from 'caminho-do-seu-arquivo-de-utilitarios';

const formattedCnpj = formatCnpj('12345678000195');

console.log(formattedCnpj); // '12.345.678/0001-95'
```

## Descrição

A função `formatCnpj` utiliza a biblioteca `@brazilian-utils/format-cnpj` para formatar uma string de CNPJ no padrão brasileiro `XX.XXX.XXX/XXXX-XX`. Esta função é útil para garantir que o CNPJ seja exibido de maneira consistente e legível.
