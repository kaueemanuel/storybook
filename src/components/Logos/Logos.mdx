# Logos

## Usage

<p>
  To make use of the Plataforma Grupo Voalle logos, simply import your svg and use it in an HTML `<img>` tag.
</p>

```tsx
import { EllevenDarkLogoSVG } from "pgv-lib/ui/logos"
;<img src={EllevenDarkLogoSVG} alt="elleven-logo" />
```

```tsx
import { GrupoVoalleLogoSVG } from "pgv-lib/ui/logos"
;<img src={GrupoVoalleLogoSVG} alt="voalle-logo" />
```
